# Vite + React

使用 React18框架 + Vite构建 + Tailwindcss样式库 进行开发。
使用 dayjs时间处理 + lodash js实用工具库 辅助开发。

#### 温馨提示使用vscode插件方便开发
1. Tailwind CSS IntelliSense : Tailwind类名提示
2. Tailwind Fold : 类名隐藏
...
